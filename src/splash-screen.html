Thank you for installing OhMyMock!

Let's get started

First step is to pin OhMyMock so the icon is always visible

Now you'll see the OhMyMock icon in an inactive state

The OhMyMock window will popup if you click the icon

Click the "enable" toggle, and you'll notice that the OhMyMock icon has changed

To see OhMyMock in action navigate to the demo site
OhMyMock is already equiped with demo data for this domain

You can click on a row to see the responses for that endpoint. In there you can modify the data and headers if you like
